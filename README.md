# ActiveTorrent
Sees which files need seeders in private trackers

## Requirements
* Python3
* Firefox
* Geckodriver (ensure that executable is in main dir)
* Selenium
* BeautifulSoup4

## Windows
```console
git clone https://github.com/nid666/ActiveTorrent.git
pip3 install selenium
pip3 install bs4
pip3 install lxml
cd ActiveTorrent
py main.py
```
